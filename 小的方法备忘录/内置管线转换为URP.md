如何把内置管线转换URP 



1.window  ==> package manager ==> 安装Universal RP 

2. Create ==> Rendering ==> URP Asset(with Universal Renderer 带通用渲染器)
3. Edit ==> Project Settings ==> Graphics =>Scriptable Render Pipeline Settings设置为创建的URP Asset
4. window ==> Rendering ==> Render Pipeline Converter 
5. 勾选Material Upgrade ==> initialize and Convert
