# Unity打字机效果实现方法

#### 1.协程逐字显示（最简单）

优点：实现简单，易控制速度

缺点：频繁字符串拼接有GC开销，不适合超长文本

```c#
public Text _text;
private Coroutine = _coroutine;

//更新文字
private void OnTextUpdated()
{
  string test = "测试简单版本的打印机效果。"
  if(_coroutine != null) StopCoroutine(_coroutine);
  _coroutine = StartCoroutine(TypeText(test));
}

private IEnumerator TypeText(string fullText)
{
  string current = "";
  foreach(char c in fullText)
  {
    current += c;
    _text.text = current;
    yield return new WaitForSeconds(0.05f);
  }
}
```





#### 2.StringBuilder优化版

相对于简单版本，用StringBuilder代替字符串拼接，减少GC压力。

```c#
public Text _text;
private Coroutine = _coroutine;
StringBuilder sb = new StringBuilder();

//更新文字
private void OnTextUpdated()
{
  string test = "测试简单版本的打印机效果。"
  if(_coroutine != null) StopCoroutine(_coroutine);
  _coroutine = StartCoroutine(TypeText(test));
}

private IEnumerator TypeText(string fullText)
{
  sb.Clear();
  foreach(char c in fullText)
  {
    sb.Append(c);
    _text.text = sb.ToString();
    yield return new WaitForSeconds(0.05f);
  }
}
```





#### 3.TextMeshPro (maxVisibleCharacters)

TextMeshPro 内置属性，性能最佳，无需重新赋值字符串

优点：性能最好，支持富文本标签，动画最流畅

```c#
public TMP_Text _text;
private Coroutine = _coroutine;

//更新文字
private void OnTextUpdated()
{
  string test = "测试简单版本的打印机效果。"
  if(_coroutine != null) StopCoroutine(_coroutine);
  _coroutine = StartCoroutine(TypeWriter(test));
}

IEnumerator TypeWriter(string fullText)
{
  _text.text = fullText;
  _text.maxVisibleCharacters = 0;
  _text.ForceMeshUpdate();
  
  for(int i = 0; i<=_text.textInfo.characterCount; i++)
  {
    _text.maxVisibleCharacters = i;
    yield return new WaitForSeconds(0.05f);
  }
}

```
